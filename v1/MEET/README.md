# 视频会议

### 简介

多人音视频可以实现一对一单聊和多人群聊，适用于社交、会议、在线教育、培训等场景。

### Demo体验

请根据需求选择渠道安装，安装完成后，可体验多人音视频功能。

- [Android Demo下载](https://www.pgyer.com/anyRTC_Meeting)
